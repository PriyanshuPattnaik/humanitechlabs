# Humanitech Labs

An innovation lab building ethical AI and open-source solutions for humanitarian impact.

## About

Humanitech Labs is dedicated to creating technology that bridges gaps between communities and opportunities, problems and solutions. We focus on research, open-source projects, and services that empower NGOs through technology.

## Tech Stack

- **Framework**: SvelteKit 2.x
- **Language**: TypeScript
- **Styling**: Custom CSS with Glassmorphism
- **Icons**: Lucide Svelte
- **Deployment**: Vercel

## Getting Started

### Prerequisites

- Node.js >= 18.0.0
- npm or pnpm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is configured for deployment on Vercel using `@sveltejs/adapter-auto`.

### Deploy to Vercel

1. Push your code to GitHub
2. Import your repository in Vercel
3. Vercel will automatically detect SvelteKit and deploy

Or use the Vercel CLI:

```bash
npm i -g vercel
vercel
```
